Data Entry Basics:

SVC input forms are color coded to repersent the type of service that was provided. Pink forms are for telephone services, yellow is for in-person services and green is for non-victim services.

The required data entry portal for your form can be found on the screen to the left. Select the form that matches the type of form you're entering. (Pink = telephone, yellow = in-person, green = non-victim).
